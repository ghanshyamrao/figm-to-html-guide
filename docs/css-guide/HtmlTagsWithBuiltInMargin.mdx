# HTML Tags with Built-in Margin and Padding

In web design, various HTML tags come with built-in margins and padding applied by default by the browser. Understanding these default styles is crucial for creating consistent and well-designed layouts. Below is a list of common HTML tags that have built-in margins and/or padding and examples of how you can customize their styles using CSS.

## Common HTML Tags with Built-in Margin and Padding

### `<body>`

The `<body>` tag often has default margins applied by browsers. You can adjust it using CSS:

```css
body {
  margin: 0;
  padding: 0;
}
